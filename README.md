# Spelling Bee Solver (C)

This project is a C program that solves the Spelling Bee game by finding all valid words from a dictionary based on given letters.

# Features
- Build dictionary using dynamic memory
- Checks required letter condition
- Filters valid words using hive letters
- Supports brute force and optimized search

# Concepts Used 
- Pointers and dynamic memory
- Strings and array
- Recursion
- Binary Search
- File i/o

# How to Run 

Compile:
make

Run:
./spellingBee.exe
